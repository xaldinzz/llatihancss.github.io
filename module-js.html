Teks & Catatan

Setiap dokumen HTML harus dimulai dengan deklarasi ``. Ini memberitahu browser bahwa halaman tersebut adalah halaman HTML5.

Contoh Kode HTML

<!DOCTYPE html>
<html>
<head>
    <title>Judul Halaman</title>
</head>
<body>
    <h1>Ini adalah Heading</h1>
</body>
</html>

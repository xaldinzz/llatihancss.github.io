Struktur Dokumen HTML

Video Tutorial

Teks & Catatan

Setiap dokumen HTML harus dimulai dengan deklarasi ``. Ini memberitahu browser bahwa halaman tersebut adalah halaman HTML5. Elemen `` adalah elemen root dari setiap halaman HTML.

Contoh Kode (Code Snippet)

<!DOCTYPE html>
<html>
<head>
    <title>Judul Halaman</title>
</head>
<body>
    <h1>Ini adalah Heading</h1>
    <p>Ini adalah paragraf.</p>
</body>
</html>

Latihan Interaktif

Coba tulis ulang kode di atas di editor teks Anda dan simpan sebagai file `.html`, lalu buka di browser!

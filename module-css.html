Materi: CSS Lengkap - Selector & Box Model

Video Tutorial CSS

Teks & Catatan

CSS Selector digunakan untuk "menemukan" (atau memilih) elemen HTML yang ingin Anda beri gaya. Box model adalah konsep penting yang menggambarkan layout elemen sebagai kotak dengan: Margin, Border, Padding, dan Content.

Contoh Kode CSS

/* Ini adalah contoh selector */
p {
  color: blue;
  font-size: 16px;
}

/* Ini adalah contoh box model */
.card {
  padding: 20px;
  border: 1px solid #ccc;
  margin: 10px;
}
